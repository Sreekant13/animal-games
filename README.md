## Games

### 🏙️ Bamboo Crisis — Conservation Strategy
Manage a panda sanctuary under siege from urban encroachment. Plant bamboo corridors, build barriers, negotiate with developers, and survive random crises to grow your panda population.

**Win:** Reach 15 pandas alive after 60 months.

| Feature | Details |
|---|---|
| City AI | Expands from all edges every month, targets undefended bamboo |
| Panda AI | Named pandas wander, eat, stress, breed, and age autonomously |
| Seasons | Spring/Summer/Autumn/Winter affect hunger, breeding, and crisis rates |
| Crises | 8 random events — disease, poachers, floods, developers, documentary crews… |
| Research | Unlock Panda Resilience, Eco-Tourism, or Corridor Growth |
| Sound | Web Audio API synthesized effects (toggle on/off) |
| Speed | 1× / 2× / 3× game speed |
| Save | Auto-save every 5 months via localStorage |

**Controls:**
- Click map cells to plant bamboo or build barriers
- `P` — Pause / Resume
- `Esc` — Cancel action mode

### 🎋 Panda's Bamboo Feast — Arcade
Move your panda left and right to catch falling bamboo stalks. Build combos, grab bonus stars, avoid hazards. Speed increases with each level.

**Controls:** `← →` Arrow Keys or `A / D`
